# FinovaAI

FinovaAI is a light fintech portfolio intelligence platform that turns broker portfolio PDFs into a clean investment dashboard using AI, LLMs, RAG, and agentic workflows.

Users can upload portfolio statements, view holdings, asset allocation, sector exposure, P&L, risk score, market updates, and AI-driven portfolio insights. The project showcases a practical full stack finance product with AI-powered data extraction, retrieval-based market context, and decision support.

## Features

- PDF portfolio upload and holdings extraction
- Portfolio dashboard with value, P&L, risk score, and allocation views
- Sector exposure and asset breakdown charts
- Portfolio stress testing for market scenarios
- Market pulse and financial news updates
- LLM-powered finance assistant
- RAG-based market and portfolio context retrieval
- Agentic AI workflow for portfolio analysis and recommendations
- Secure authentication-ready user flow
- Light, responsive fintech UI
